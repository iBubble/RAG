担保条款专项审查清单

加载条件：C档合同审查时自动加载，或合同中涉及保证、抵押、质押、定金等担保条款时加载。
法律依据：《中华人民共和国民法典》担保制度及《最高人民法院关于适用〈中华人民共和国民法典〉有关担保制度的解释》（2021年）。
说明：本模块通篇为担保的法定成立要件、登记/交付、决议等高漏检技术点——正是模型容易一带而过、必须逐项核对的增量，故整体保留。

一、保证条款审查
- [ ] 保证方式：一般保证还是连带责任保证。《民法典》第686条：约定不明的按一般保证承担责任。审查是否明确约定连带责任保证
- [ ] 保证期间：是否明确约定，未约定的法定保证期间为主债务履行期限届满之日起六个月
- [ ] 保证人资格：是否具备代偿能力，是否为机关法人、公益性事业单位等禁止担保的主体
- [ ] 保证范围：是否涵盖主债权、利息、违约金、损害赔偿金、实现担保物权的费用

二、公司对外担保决议审查
- [ ] 公司为他人提供担保：是否经公司章程规定的机构（董事会或股东会）决议
- [ ] 公司为其股东或实际控制人提供担保：必须经股东会或股东大会决议，且该股东或实控人不得参加表决
- [ ] 上市公司对外担保：需股东大会决议
- [ ] 审查决议文件的真实性、程序合规性
- [ ] 我方作为相对人接受担保时，应审查并留存对方的决议文件

三、抵押条款审查
- [ ] 抵押物是否属于法律允许抵押的财产
- [ ] 是否需要办理抵押登记（不动产抵押、动产浮动抵押等未登记不得对抗善意第三人）
- [ ] 抵押物上是否存在在先权利负担（查封、在先抵押等）
- [ ] 抵押权实现方式是否约定（折价、拍卖、变卖）
- [ ] 最高额抵押的债权确定期间和最高债权额

四、质押条款审查
- [ ] 动产质押：是否约定交付质押财产的时间和方式
- [ ] 权利质押：是否需要办理出质登记（股权、知识产权、应收账款等）
- [ ] 质押财产的保管责任和风险承担

五、定金条款审查
- [ ] 定金数额是否超过主合同标的额的20%（超过部分不产生定金效力）
- [ ] 定金罚则是否明确约定（给付定金方违约丧失定金，收受定金方违约双倍返还）
- [ ] 定金与违约金竞合时的选择适用约定

六、我方立场审查要点（担保特有的立场区分）
如我方为担保提供方（保证人/抵押人/质押人）：
- [ ] 担保范围是否过宽，是否约定了合理上限
- [ ] 反担保措施是否到位
- [ ] 担保解除条件是否明确

如我方为担保接受方（债权人）：
- [ ] 担保是否充分覆盖债权
- [ ] 担保物权的设立是否完成（登记/交付）
- [ ] 担保人的资信和代偿能力
- [ ] 在合同中要求担保人提供决议文件并作为附件
