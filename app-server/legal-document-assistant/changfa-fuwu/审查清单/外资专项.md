外资企业合同专项审查清单

加载条件：客户档案中 客户类型=外资企业 时自动加载。
使用方式：通用审查要点的叠加模块，在完成通用审查的基础上额外审查以下专项要点。
设计原则：只列外资企业特有、模型不会主动想到的增量（准入、外汇、数据出境、涉外文件、涉外争议解决）；通用质量判断交由通用清单。
法规版本提示：本清单点出适用的法规框架，但具体版本/条文以审查时在线检索为准（如外商投资准入负面清单、数据跨境相关规定均定期更新），不得凭记忆套用，存疑标【待查】。

一、外商投资准入（外资特有，模型不会主动想到）
- [ ] 合同涉及的经营领域是否触及外商投资准入负面清单（最新版）的限制类、禁止类，是否有股比或高管要求 [全档]
- [ ] 是否涉及需前置审批、行业许可或安全审查的领域 [B/C档]

二、跨境资金与外汇（外资特有）
- [ ] 涉及跨境收付款的，关注外汇登记、结售汇、资本项目外汇管理合规，境外付款路径是否合法 [B/C档]
- [ ] 跨境交易的预提所得税、税收协定待遇是否已考虑 [B/C档]

三、数据与个人信息出境（缺失检查）
- [ ] 涉及数据或个人信息出境的，检查是否缺少数据出境安全评估、个人信息出境标准合同备案或认证等合规安排（个保法、数据安全法及数据跨境相关规定，适用版本待查）[B/C档]

四、涉外主体与文本（缺失检查/高漏检）
- [ ] 外方主体的境外主体资格、签约授权文件是否经公证认证或附加证明书（海牙认证），我方是否留存 [全档]
- [ ] 中外文文本是否约定以某一语言版本为准；缺失的，检查并提示补充 [全档]

五、适用法律与争议解决（外资特有的涉外考量）
- [ ] 约定适用境外法律或提交境外仲裁/法院的，关注是否违反中国强制性规定、裁决或判决在中国的承认与执行 [B/C档]
- [ ] 管辖/争议解决的基本偏好仍按通用清单（我方所在地）；涉外可评估约定境内仲裁机构进行涉外仲裁 [B/C档]
