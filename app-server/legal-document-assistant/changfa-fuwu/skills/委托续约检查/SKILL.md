# 委托续约检查

你在帮中国律师检查常法委托合同的到期情况。

## 执行步骤

### 第一步：读取委托合同台账
读取工作根目录下的 `委托合同台账.xlsx`（位置可从律师档案的工作根目录推断，或询问用户）。台账列：客户简称、委托合同全称、合同文件路径、服务起始日、服务到期日、年费金额、付款方式、客户对接人、备注。
如台账不存在，扫描工作根目录下各客户文件夹中名称含"委托合同""顾问合同"的文件，尝试提取服务期限，协助用户建立初始台账。

### 第二步：计算到期状态
用 datetime.now() 获取当前真实日期为基准，逐客户计算距到期天数，分级：
- 已过期：到期日 < 今天 →"已过期X天，目前处于无合同服务状态，请尽快联系续约"
- 紧急：到期日 ≤ 今天+30天 →"将在X天内到期，请立即联系续约"
- 关注：到期日 ≤ 今天+60天 →"将在X天内到期，建议开始准备续约"
- 正常：到期日 > 今天+60天 →"距到期还有X天"

### 第三步：输出提醒清单
按紧急程度排序输出，分【已过期】【紧急】【关注】【正常】四组，每条含客户、剩余/超期天数、到期日、年费、对接人。末尾给出统计：共X个客户，已过期X、紧急X、关注X、正常X。
