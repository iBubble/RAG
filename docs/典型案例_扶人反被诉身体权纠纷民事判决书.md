# 典型案例：王某某与李某民事侵权身体权纠纷民事判决书

原告：王某某，女，72岁，居民。
被告：李某，男，11岁，学生。监护人：李某父。

【案情简述】
原告王某某在中山路十字路口摔倒受伤。原告主张系被告李某骑行自行车将其撞倒；被告李某及监护人主张是李某见老人摔倒后好心上前扶起，并非李某撞倒。

【法院查明】
事发后李某家属曾向王某某家属支付了2000元“医疗慰问金”，并在一份写有“如确系撞伤后续医药费再协商”的便条上签字。虽无直接监控，但结合事后便条及垫付医药费行为，存在高度盖然性。

【法院认为】
被告李某的行为与原告王防某摔倒受伤之间存在因果关系。结合原告王某某自身患有严重骨质疏松，摔倒导致的骨折后果中，自身身体状况原因力占50%。李某系限制民事行为能力人，其侵权责任由监护人承担。

【判决结果】
被告李某的监护人于本判决生效之日起十日内赔偿原告王某某医疗费、护理费等各项损失共计人民币 24,500 元。
