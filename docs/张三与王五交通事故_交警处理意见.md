# 道路交通事故认定书（简易程序）

编号：第20260601-009号

## 一、当事人基本情况
- **当事人甲（受害人）**：张三，男，35岁，骑行自行车。
- **当事人乙（肇事人）**：王五，男，40岁，驾驶闽D88888轻型轿车。

## 二、道路交通事故事实
2026年6月1日09时05分许，王五驾驶闽D88888轿车在人民路由东向北右转弯时，未注意观察右侧非机动车道，与同向直行由张三骑行的自行车发生擦碰，造成张三跌倒受伤、两车不同程度损坏。事故发生后，王五驾驶轿车驶离现场，未保护现场及抢救伤者，构成交通事故逃逸行为。

## 三、交通事故原因及责任认定
- **王五**：驾驶机动车转弯未让直行的车辆通行，违反《中华人民共和国道路交通安全法实施条例》第五十一条之规定；且在发生事故后逃逸，违反《中华人民共和国道路交通安全法》第七十条第一款之规定。
- **张三**：正常骑自行车直行，无交通违法行为。

根据《中华人民共和国道路交通安全法实施条例》第九十二条之规定：“发生交通事故后当事人逃逸的，逃逸的当事人承担全部责任。”

**认定结论**：王五承担本次道路交通事故的**全部责任**，张三**无责任**。
