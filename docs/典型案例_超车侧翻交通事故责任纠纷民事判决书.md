# 典型案例：赵某与张某某机动车交通事故责任纠纷民事判决书

原告：赵某，男，65岁，居民。
被告：张某某，男，23岁，外卖骑手。

【案情简述】
被告张某某骑行二轮燃油摩托车（送外卖途中）在建设路由东向西超车时，由于超车距离过近，导致在其右侧骑行电动自行车的原告赵某受惊吓自行车晃动并发生侧翻摔倒受伤。

【法院查明】
交警部门出具的事故证明无法确定碰撞痕迹。但事故现场勘验及视频录像显示，张某某摩托车在超车时时速达到55km/h，且距离电动车不足半米。

【法院认为】
虽然两车未发生直接接触，但张某某在狭窄路段高速超车且未保持安全距离，其危险驾驶行为对赵某造成了惊吓并诱发侧翻。这在民事侵权中属于无接触交通事故。根据高度盖然性原则，超车行为与摔倒存在因果关系。

【判决结果】
被告张某某承担本次事故70%的主要侵权责任，赔偿原告赵某各项人身损害损失合计 58,000 元。
